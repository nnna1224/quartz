SSHを簡単にするためのショートカット設定方法
ssh接続をするには，`ssh <ユーザ名>@<ホスト名>`などを毎回打つ必要があり，大変である．
この記事では，接続先のショートカットを設定することで，少しだけコマンドを省略できるように設定する．

`~/.ssh/config`に，以下のようなことを記述する．
code:~/.ssh/config
 Host <任意の接続名(hoge)>
     HostName <ホスト名>
     User <ユーザー名>
     Port <ポート番号>
     IdentityFile <鍵へのPATH>
     ServerAliveInterval 60
※PortやIdentityFile，ServerAliveIntervalはオプション

すると，以下の様にsshのコマンドを省略することが出来る．
code:terminal内で実行
 ssh <任意の接続名(hoge)>


例(KKIに接続する場合)：
code:~/.ssh/config
 Host kki
 	HostName jw01.kki.yamanashi.ac.jp
 	User t22cs000
code:SSH接続(設定した場合)
 ssh kki
code:設定しない場合
 ssh t22cs000@jw01.kki.yamanashi.ac.jp
