KKIの適当な場所で以下を実行する．
```shell
~/../t22cs040/share/setweb.sh
```

ファイルの中身は一応以下の様になっている(セキュリティ的に心配なら手順(https://lww.kki.yamanashi.ac.jp/~t23cs014/#KKI-user)に沿って環境構築すること)．
※ 実際のファイルの中身は`cat ~/../t22cs040/share/setweb.sh`で確認可能．
```shell
#!/usr/bin/sh
set -eu

HTML_TEMPLATE='<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Template</title>
</head>
<body>
  <p>
  Hello from Template!!<br/><br/>
  This file was automatically generated by a script created by T22CS040.<br/>
  Enjoy creating your website!!
  </p>
</body>
</html>'

for dir in local_html public_html; do
  target="$HOME/$dir"

  if [ ! -d "$target" ]; then
    mkdir -p "$target"
    chmod 711 "$target"
    echo "Created directory $target"
  else
    echo "Directory exists, skipped: $target"
  fi

  if [ ! -f "$target/index.php" ] && [ ! -f "$target/index.html" ]; then
    printf '%s\n' "$HTML_TEMPLATE" > "$target/index.php"
    chmod 744 "$target/index.php"
    echo "Created index.php in $target"
  else
    echo "Skipped index.php creation in $target (index.php or index.html exists)"
  fi

  if [ ! -d "$target/.git" ]; then
    git init "$target" > /dev/null
    echo "Initialized Git repository in $target"
  else
    echo "Git repository already exists in $target, skipped"
  fi
done
```
