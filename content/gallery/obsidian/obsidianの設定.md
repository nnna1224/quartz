## 前提
新規にobsidianを使い始める場合は，[[obsidianのインストールとvaultの作成]]を先に行う．

## 設定画面への移行
vaultを作成するか既存のvaultを開くと，以下に示す画面が表示される．
ここで，左下のvault名の右側にある歯車(設定)マークを押して，設定画面を開く．
![[obsidian-app-start.png]]

## 私の行っている設定
**設定項目**
【ファイルとリンク】
- 新規ノートの作成場所を「desk」に変更．
- 新規添付ファイルの作成場所を「現在のフォルダのサブフォルダ」に変更．
	- サブフォルダの名称を「attachments」にする．

【コアプラグイン】
- 同期は切っておく(勝手に使われたら面倒なので)．

【コミュニティプラグイン】
- コミュニティプラグインの制限モードの表示が「有効化」になっていることを確認する．
	- コミュニティプラグインを無効化するには，これを有効化するというややこしい構造になっている．
	- 要するに，ここは「制限モード」の設定で，制限が無効になっている状態にするということ．
- コミュニティプラグインの「閲覧」をクリックして，コミュニティプラグインを確認する．

**コアプラグインの設定**
【デイリーノート】
- 新規ファイルの場所を「daily」にする．

**インストールするコミュニティプラグイン**
- Calendar
- Git

**コミュニティプラグインの設定**
【Calendar】
- デフォルトで使用．

【Git】
- Auto commit-add sync interval (minutes)を「5」に設定．
- Pull on startupのトグルをオン．
- Custom Git directory pathを「..\」に変更．
	- Quartzとの兼ね合いのため．gitのルートとvaultが一致している場合は設定不要．
